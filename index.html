topic 6. 데이터타입- 문자열과 숫자
data type에서 number는 숫자, string은 문자열을 말한다.
간단하게 결과를 알아볼 때는 alert속성을 안쓰고 그냥 해도된다.
number data type에서 +는 좌,우의 값을 더해서 하나의 값을 만드는 이항연산자이고 산술연산자이다. (+,-,*,/)
문자는 큰따옴표에서 큰따옴표("")로, 작은따옴표에서 작은따옴표('')로 끝나면된다.
문자를 처리할 때 사용하는 명령들을 찾아보려면 javascript string이라고 친다.
string,number에 사용하는 명령들은 굉장히 많은데 이 명령들의 문법에 대해서 숙지하고 오타없이 console에나 sheet에 작성하면 실행된다.
중요한 것은! number data type은 1+1=2라는 결과를 실행하지만 string data type은 "1"+"1"="11"이라는 결과를 실행한다는 것!
데이터에 따라서 추출되는 결과가 다르기때문에 분류를 잘해야한다.
경험으로 이해하는 것보다 익숙해지는 것이 중요!
-이번에 배운 것

  • str.length는 길이를 나타내는 명령
  • str.toUpperCase()는 대문자로 바꿔주는 명령
  • str.indexOf('str')는 해당문자열의 순서를 찾아주는 기능이다.
